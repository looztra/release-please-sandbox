workflow. They are useful for historical context, but the top-level
[docs](../README.md) pages are the reference documentation for day-to-day use.

- [Release tag options](release-tag-options.md): alternatives considered for
  making the release tag point at the staging-deployed commit.
- [Release workflow proposal](release-workflow-proposal.md): original release
  workflow proposal, review revisions, failure modes, and caveats.
- [Staging deploy gating study](staging-deploy-gating-study.md): alternatives
  and race analysis for staging deploy gating.
